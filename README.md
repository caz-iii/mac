<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

Apple Mac device.

### Installation

1. Clone the repo

2. Auto deploy

The `install.sh` script will take care of updating the system and installing programs. 

```bash
cd mac
```

```bash
chmod +x install.sh
```

```bash
./install.sh
```
